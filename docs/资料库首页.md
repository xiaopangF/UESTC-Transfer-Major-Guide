# 电子科技大学转专业资料库

这是学生共同维护的非官方资料索引。仓库已收录年度附件摘要、匿名社区经验、历史数据线索和考后机考回忆；具体政策与时间请以学校及学院当年通知为准。

## 浏览资料

| 分类 | 入口 | 说明 |
| --- | --- | --- |
| 流程 | [总体流程](process/总体流程.md) · [年度流程](process/by-year/年度流程索引.md) | 将稳定框架和逐年变化的信息分开 |
| 2026 细则 | [清水河校区 16 学院摘要](process/by-year/2026-qingshuihe-college-policies.md) | 计划按 `2025级`、`2024级`标注，官网链接待补 |
| 数据 | [历年申请数据](data/历年申请数据说明.md) | 记录人数、名额、口径、来源和核验时间 |
| 机考 | [计算机专业机考回忆](exams/computer/计算机专业机考回忆索引.md) | 只收录考核结束后的考生回忆版 |
| 经验 | [面试与转专业经验](experiences/面试与转专业经验索引.md) | 按年份和目标专业整理 |

## 使用提示

- 标为“历史资料”的页面不能直接代表当前政策。
- 单人回忆可能不完整，阅读时请同时查看来源类别和核验状态。
- 发现错误或希望投稿时，请返回 [仓库首页](../README.md) 查看参与方式。
