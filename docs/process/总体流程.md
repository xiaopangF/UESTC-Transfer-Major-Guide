# 总体流程

> 本页是阅读和核验转专业资料的通用清单，不代表某一年度的具体政策。资格、计划、时间、材料、考核和录取规则均以学校及学院当年通知为准。当前资料中的官方页面尚未逐一补齐，统一标记为“官网链接待补”。

## 先确认年份与年级口径

转专业文件常同时出现自然年、学年学期和学生年级，三者不能混用。例如，2026 年发布的细则可能分别面向 **2025级** 或 **2024级** 学生。阅读任何计划数或条件前，应先记录：

- 文件发布或实施的自然年；
- 适用年级，统一写成“2025级”“2024级”等形式；
- 校区、批次、目标学院和目标专业；
- 计划数的单位及含义，区分接收计划、进入考核人数和最终录取人数；
- 官方通知标题、官网链接和最后核验日期。

若来源没有写清年级，不根据自然年自行推断，标记为“适用年级待核实”。本仓库涉及的 2026 年接收计划均在对应页面逐项标明“2025级计划”或“2024级计划”；无法确认的数字不录入为确定事实。

## 1. 查找当年通知

先查学校层面的工作通知，再查目标学院实施细则、附件和后续补充通知。同一学院若发布更正或补充文件，应同时保留原通知和更新说明。

核对项目：

- 发布单位、标题、发布日期和适用对象；
- 校区、年级、批次及报名入口；
- 学校通知与学院细则是否互相引用；
- 是否存在更新版本、附件或结果公示；
- 官网链接及最后访问日期。

当前入口：**官网链接待补**。

## 2. 确认申请资格

逐项对照校级通知和目标学院细则，不把其他学院或往年经验套用到当年。常见核对项包括：

- 学籍状态、当前年级和所在校区；
- 原学院、原专业或招生类别限制；
- 必修课程、学分、成绩或不及格课程要求；
- 体检、纪律处分或其他资格条件；
- 可填报的志愿数量、是否允许跨类申请及志愿优先级；
- 不符合条件时的咨询、异议或复核渠道。

社区经验只能帮助理解准备过程，不能替代资格审核结论。

## 3. 核对接收计划

查看计划时，把数字和口径写在同一句中，例如“2025级计划：若干人”或“2024级计划：若干人”。还应记录：

- 计划对应的学院、专业、方向和年级；
- 名额是否按年级、专业或批次分别设置；
- 是否注明计划可根据报名、考核或培养条件调整；
- 数字是接收计划、资格审核通过人数、考核人数还是录取人数；
- 数据来自官方文件还是社区回忆。

未找到公开官网页面时保留“官网链接待补”，不以本地附件代替可公开访问的来源。

## 4. 准备并提交申请

以当年通知为准确认报名系统、开放时间和截止时间。提交前建议检查：

- 系统中的年级、专业和联系方式是否准确；
- 志愿、申请理由及学院要求的材料是否完整；
- 成绩单、证明或作品等材料的格式和提交方式；
- 是否允许在截止前修改或撤回；
- 提交后是否有成功状态、回执或待审核提示。

不要在公开投稿中上传含姓名、学号、联系方式或系统截图的原始申请材料。

## 5. 等待资格审核

资格审核通常先于考核，但审核主体、结果渠道和复核方式随年度及学院变化。应记录：

- 审核结果发布位置和查询方式；
- 补充材料或异议提交的截止时间；
- 进入考核名单的统计口径；
- 后续通知是否要求再次确认参加考核。

“报名成功”不等于“资格审核通过”，通过审核也不等于获得录取资格。

## 6. 参加考核

不同学院可能采用材料审核、笔试、机考、面试或组合考核。参加前按当年细则确认：

- 日期、地点、签到和身份核验要求；
- 考核科目、时长、允许携带的物品或可用工具；
- 各环节分值、权重、合格线及缺考处理；
- 设备、编程环境或材料展示要求；
- 临时调整的官方通知渠道。

本仓库的机考内容均为考试结束后的考生回忆摘要，不是官方题面或题库，也不能据此推定下一年度考法。

## 7. 查看录取与公示

录取规则应按学院当年细则原意摘要，特别注意总成绩计算、分年级或分专业排序、合格条件以及同分处理。查看结果时核对：

- 公示名单、期限和发布单位；
- 拟录取、正式录取等状态的区别；
- 是否需要在规定时间内确认、放弃或办理手续；
- 成绩复核、异议或申诉渠道；
- 计划调整后最终录取人数的口径。

社区中的排名或分数叙述不用于推导固定录取线。

## 8. 办理后续手续

录取后仍需关注学籍、培养方案和课程衔接。常见事项包括：

- 学籍异动和新专业报到；
- 已修课程认定、替代或补修；
- 培养方案、选课、班级和教材安排；
- 宿舍、校区或学费变化（如适用）；
- 奖助、推免或其他后续政策的适用规则。

这些事项由相应管理部门和学院解释，历史经验仅供提前准备。

## 如何使用年度资料

具体年份的通知、计划和学院差异收录在 [年度流程](by-year/年度流程索引.md)。阅读时按以下证据等级判断：

| 标记 | 含义 | 使用方式 |
| --- | --- | --- |
| `official` | 已有学校或学院公开页面支持 | 仍需检查发布日期和适用年级 |
| `corroborated` | 至少两个独立来源互相印证 | 不代替官方通知 |
| `single_report` | 单份社区整理或单人回忆 | 仅作经验参考 |
| `unverified` | 来源或口径尚未完成核验 | 不作为确定事实使用 |

新年度政策应新增年度页面，不用新政策覆盖旧记录；官网链接补齐后，应同步更新来源登记和最后核验日期。
