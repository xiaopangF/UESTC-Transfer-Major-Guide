# 资料来源登记

本页登记本轮整理使用的三个本地来源文件。SHA-256 用于识别整理时读取的具体字节副本，不能单独证明文件来自官网、内容完整或属于最终版本。三个原始二进制均保留在仓库外，Git 只收录经过改写、脱敏和核验标注的 Markdown 摘要。

## 处理原则

- 不提交 DOC、DOCX、RAR、截图、嵌图或临时解包文件。
- 官方附件未找到公开页面时，只能标为“附件副本待官网核验”，不能仅凭文件外观认定为官网最终版。
- 社区经验与考生回忆只保留必要事实、考点和方法摘要，不搬运长段原文或完整题面。
- 姓名、QQ、其他社交账号、群信息、电话、邮箱、精确个人排名、精确个人成绩和可识别的经历组合不进入仓库。
- 文档作者、最后修改者、个人文件名、修改时间等文件属性元数据不进入公开摘要。
- 当前所有待补官方页面统一标记为：**官网链接待补**。
