# 内容政策

## 来源与真实性

- 资格、日期、名额、考核方式等可核实事实应附公开来源。
- 官方信息、多人交叉印证、单人自述和未核实信息必须明确区分。
- 引用只保留说明事实所需的少量文字，优先链接原页面。
- 页面被合并只代表完成基本审核，不代表仓库保证其完整或正确。

## 版权

- 只提交本人原创内容，或有明确再发布授权的内容。
- 对许可证不明确的公告、附件、图片和题目，只记录标题、链接与必要摘要。
- 不复制付费课程、付费题库、群文件或其他受限制材料。
- 外部内容仍归原权利人所有，不受本仓库许可证覆盖。

## 隐私

- 默认不收集真实姓名、学号、联系方式、精确班级、证件、照片和人脸信息。
- 不描述足以识别具体教师、工作人员或其他考生的细节。
- 投稿第三人的经历前必须获得对方同意。
- 详细要求与删除方式见 [隐私说明](../../PRIVACY.md)。

## 表述

- 区分“官方要求”“个人经历”“个人建议”和“推测”。
- 不把相关性写成因果，不承诺录取结果。
- 存在多个公开口径时同时保留，并解释差异。
- 纠错应修改原页面并保留来源，不用无来源的新说法直接覆盖。
