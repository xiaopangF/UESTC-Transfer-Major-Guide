# 机考材料规范

## 可以提交

- 对应考核完全结束后，考生凭记忆用自己的语言整理的内容。
- 作为同一篇回忆记录组成部分的考试环境、题型与准备方式复盘。

## 不可以提交

- 仍在进行或尚未结束的考试内容。
- 内部题库、泄露试卷、绕过权限获取的页面或文件。
- 考试平台账号、访问令牌、抓取脚本或规避监考的方法。
- 未经许可的扫描件、截图、付费资料复制品或大段转载。
- 含有考生、教师或工作人员个人信息的材料。

## 标注要求

每篇资料必须标注年份、目标学院与专业、记忆置信度和核验状态。考生回忆版可能存在缺题、错序和措辞偏差；投稿者整理的解答只能称为“参考思路”，不能冒充官方答案。

机考回忆中附带的代码只作为文本展示，不会被资料校验流程执行。需要说明代码时优先使用短小、静态的代码块，不上传二进制文件。
