# 电子科技大学转专业资料库

这是学生共同维护的非官方资料索引。当前仓库只提供整理结构和投稿入口，具体政策与时间请以学校及学院当年通知为准。

## 浏览资料

| 分类 | 入口 | 说明 |
| --- | --- | --- |
| 流程 | [总体流程](process/general.md) · [年度流程](process/by-year/index.md) | 将稳定框架和逐年变化的信息分开 |
| 数据 | [历年申请数据](data/index.md) | 记录人数、名额、口径、来源和核验时间 |
| 机考 | [计算机专业机考回忆](exams/computer/index.md) | 只收录考核结束后的考生回忆版 |
| 经验 | [面试与转专业经验](experiences/index.md) | 按年份和目标专业整理 |

## 使用提示

- 标为“历史资料”的页面不能直接代表当前政策。
- 单人回忆可能不完整，阅读时请同时查看来源类别和核验状态。
- 发现错误或希望投稿时，请返回 [仓库首页](../README.md) 查看参与方式。
