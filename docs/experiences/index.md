# 面试与转专业经验

本目录按考核年份整理面试、准备过程和完整转专业经历。经验是个人样本，不代表当年全部情况，也不能预测后续年份。

## 阅读标识

- `single_report`：一位投稿者的自述。
- `corroborated`：已与至少一个独立来源交叉核对。
- `official`：仅用于有官方链接支持的流程事实，不用于给个人建议背书。
- `unverified`：暂未完成来源或格式核验。
