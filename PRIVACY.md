# 隐私说明

本仓库只收集完成资料整理所必需的信息，不要求贡献者公开真实姓名、学号、联系方式或精确成绩。

## 投稿者须知

GitHub 仓库、Issue、Pull Request 和提交记录默认是公开的。正文使用“匿名贡献者”署名，并不能隐藏 GitHub 用户名、活动记录或提交邮箱。投稿前建议：

- 在 GitHub 邮箱设置中启用隐私保护，并使用 GitHub 提供的 noreply 邮箱提交。
- 删除截图和文档中的姓名、学号、二维码、人脸、文件属性等识别信息。
- 将绩点、排名、原专业班级等改写为足够宽的区间。
- 不提交第三人的经历，除非已获得对方明确同意。

仓库默认优先接收纯文本。图片、PDF、压缩包和可执行文件通常不接受；确需引用时请提供合法的公开来源链接。

## 删除与更正

公开的普通错误可以通过“内容报告”或“流程与内容纠错”Issue 提交，只填写文件路径和问题类型，不要再次粘贴敏感内容。

涉及隐私泄露时，请通过仓库维护者在 GitHub 个人资料或仓库 About 区域公布的私下渠道联系。仓库正式公开前，维护者应补充一个真实可用的私下联系渠道。

维护者可以从当前版本中删除或匿名化内容，但 Git 历史、Fork、缓存和他人已有的克隆可能无法完全收回。发生严重泄露时，维护者会评估是否需要重写历史并联系 GitHub 支持。
