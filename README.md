# 电子科技大学转专业资料库

> 学生自发维护的非官方资料库。流程、资格、名额和考核方式可能逐年变化，请始终以电子科技大学及相关学院当年发布的通知为准。

本仓库用于整理电子科技大学转专业相关的公开数据、流程资料和学生经验。目前只建立了资料结构与投稿模板，不包含任何未经核实的年份数据、题目或个人经历。

## 如何投稿

- 不熟悉 Git：在 [Issues](../../issues/new/choose) 中选择“经验投稿”“机考回忆投稿”“申请数据新增或纠错”等中文表单。
- 熟悉 Git：阅读 [贡献指南](CONTRIBUTING.md)，基于 [templates](templates) 中的模板新增 Markdown，或更新数据 CSV 后提交 Pull Request。
- 只想指出错误：提交“流程或内容纠错”Issue，并附上公开来源。

所有投稿都会保留来源类别与核验状态。“已收录”只表示格式、来源和风险检查已完成，不代表本仓库为内容真实性背书。

## 内容边界

- 不使用校徽，不冒充学校、学院或任何官方组织。
- 不提交仍在进行的考试内容、内部题库、绕过授权获得的材料或未经许可的付费资料。
- 机考区只收录“考生回忆版”，不能宣称为官方真题。
- 不公开姓名、学号、电话、邮箱、社交账号、准考证号、人脸等个人信息。
- 官方公告原则上只记录标题、链接和必要摘要；许可证不明确时不搬运全文或附件。

更完整的要求见 [贡献指南](CONTRIBUTING.md)、[隐私说明](PRIVACY.md) 和 [机考材料规范](docs/policies/exam-materials.md)。
